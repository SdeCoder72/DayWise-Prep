# 🧮 Java 2D Arrays: Introduction

## 📌 What is a 2D Array?
A 2D array is an array of arrays. Think of it as a matrix with rows and columns.

```java
int[][] matrix = new int[3][4]; // 3 rows, 4 columns

```
